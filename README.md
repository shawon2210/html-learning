### 1. Image Toggle Demo
- **File**: `demo.html`
- **Description**: A simple HTML page that demonstrates toggling image visibility with JavaScript
- **Features**:

  - Centered layout with proper styling
  - Accessibility features (aria-pressed attributes)
  - Responsive design
